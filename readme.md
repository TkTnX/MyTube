# MyTube - клон YouTube на MERN (MongoDB, Express, React, Node.JS)

# Запуск проекта

### Терминал №1

1. cd ./client
2. npm i -f
3. npm run dev

### Терминал №2

1. cd ./backend
2. npm i
3. npm run dev

# Технологии

- MongoDB
- Mongoose
- Express
- svix

## Frontend

- React
- React Router Dom
- TypeScript
- React-query
- Tailwind
- Clerk
